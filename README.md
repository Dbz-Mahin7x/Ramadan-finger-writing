## 🎮 HOW TO USE

### 💻 Desktop
- **Hold `Shift`** → Draw
- **Press `Space`** → Clear canvas
- **Click colors** → Change brush color
- **Scroll wheel** → Adjust brush size
- **Save button** → Download artwork

### 📱 Mobile
- **Two-finger touch** → Draw
- **Tap colors** → Change color
- **Clear button** → Reset canvas
- **Save button** → Download

## 🚀 DEPLOY YOUR OWN

### One-Click Deploy

<a href="https://vercel.com/new/clone?repository-url=https://github.com/Dbz-Mahin7x/Ramadan-finger-writing">
  <img src="https://vercel.com/button" alt="Deploy with Vercel">
</a>

### Files Needed

```

📦 Ramadan-finger-writing/
├── 📄 index.html      # Main app (contains everything)
└── 📄 vercel.json     # Vercel config

```

### `vercel.json`

```json
{
  "name": "ramadan-finger-writing",
  "version": 2,
  "builds": [
    {
      "src": "*.html",
      "use": "@vercel/static"
    }
  ],
  "headers": [
    {
      "source": "/(.*)",
      "headers": [
        {
          "key": "Permissions-Policy",
          "value": "camera=(self)"
        }
      ]
    }
  ]
}
```
